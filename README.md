# Tasseo

Tasseo is a mystical oracle app for cup readings, symbolic interpretation, astrology, and conversational guidance.

Upload Turkish coffee, espresso, tea, latte, matcha, or similar cup photos, discover hidden symbols, and continue each reading as a personal Oracle conversation.

## Website

[Visit the Tasseo landing page](https://tassao-app.web.app)

## Highlights

- Cup Vision for coffee, tea, espresso, latte, matcha, and modern drink patterns
- Oracle Chat for follow-up questions after each reading
- Symbol Collection with hidden motifs and spiritual categories
- Astrology-based onboarding and first reading
- XP, levels, streaks, credits, and premium plans
- Multilingual and theme-aware experience
